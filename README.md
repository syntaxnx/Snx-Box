# Snx-Box
script de instalação automática de pacotes utilizado apenas para termux

----------------------//
# Instalação 

➛ apt update
➛ apt upgrade
➛ apt install git
➛ git clone https://github.com/syntaxnx/Snx-Box.git
➛ cd Snx-Box
➛ chmod +x snx.sh
➛ sh snx.sh.sh
